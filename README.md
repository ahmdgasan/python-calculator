Hello, i just made a new simple calculator with Python and i hope you like it.

To use it, follow these steps :

1. Download python from https://www.python.org/
2. Download the calculator.py file ( or you can copy paste the code )
3. Run any terminal on your pc ( PowerShell, CommandPrompt, VSCode, etc. )
4. If you use Terminal or PowerShell, type 'cd C:\Users\YourUser\Downloads'. If it said 'No such file or directory' then it means
   you download it on different location.
5. type 'python calculator.py', then it should work.
6. If you use VSCode, open the folder where you downloaded the calculator.py file and then click the play button on the right top.
